#### 安装教程

- 关联两个远程仓库

```bash
#gitee
git remote add gitee https://gitee.com/hchenp/neu-soft_-leadnews.git

#github
git remote add github https://github.com/Pxlchen/NeuSoft_Leadnews.git
```



- 克隆单一仓库

```bash
git clone https://gitee.com/hchenp/neu-soft_-leadnews.git
```



- 查看仓库名称

```bash
git remote -v
```



- 提交本地

```bash
git add .

git commit -m '提交信息'
```



- 推送

```bash
git push #名称 #分支
```
